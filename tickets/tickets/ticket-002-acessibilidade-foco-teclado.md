# Título do problema
Links e botões não exibem estado de foco ao navegar com TAB 

## Descrição
Ao navegar pela página de resultados usando a tecla TAB, alguns links de produtos e botões não exibem estado visual de foco.

## Passos para reproduzir
1.Acessar amazon.com.br.
2.Pesquisar por qualquer item (ex.: “mouse gamer”).
3.Pressionar a tecla TAB repetidamente.
4.Observar se todos os elementos clicáveis mostram o foco visual.

## Resultado atual
Alguns elementos não mostram foco visual ao navegar com TAB.

## Resultado esperado
Todos os elementos interativos devem exibir estado de foco (bordas, destaque ou alteração de cor).

## Impacto
Pode dificultar a navegação para usuários que dependem de teclado.

- Tipo: Acessibilidade
- Severidade: Baixa
- Prioridade: Baixa
